Статический сайт для ghjdtltybz викторины с выбором вариантов ответов.
Целевая аудитория: широкий круг пользователей
Стек: html+css+js.

# Conventions
- вопросы и ответы для викторины бери в resourses/
- для статической сборки используй scoped/BEM-style классы в CSS/styles.css, не создавай .styl файлы
